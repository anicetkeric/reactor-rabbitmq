# reactor-rabbitmq

RabbitMQ Producer and Consumer with Reactor RabbitMQ and Spring WebFlux

## Tech Stack

**spring-webflux**, **maven**, **java21**, **RabbitMQ**, **docker**

## Run Locally

Clone the project

```bash
  git clone https://github.com/anicetkeric/reactor-rabbitmq.git
```
